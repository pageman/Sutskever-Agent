## Known Limitations

1. **Knowledge boundary**: Agent knowledge is bounded by training cutoff. Papers
   published after the training cutoff are not covered.
2. **Hallucination risk on paper details**: Claims not grounded in paper_index.yaml
   or arxiv_fetcher results may be hallucinated. Users should verify against original papers.
3. **Mathematical correctness**: Derivations are explanatory, not formally verified.
   For production use, verify all mathematical claims against the original papers.
4. **NumPy implementation correctness**: Code is reviewed against known patterns
   (knowledge/numpy_patterns.md) but not executed. Numerical correctness is not guaranteed.
5. **GitHub API dependency**: PR review functionality requires a valid GITHUB_TOKEN
   with appropriate permissions.
